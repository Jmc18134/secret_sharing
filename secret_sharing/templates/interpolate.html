Interpolation Calculator

Given a list of n points, this calculator will use Lagrange interpolation to calculate the unique n-1 degree polynomial passing through those points.

Enter the points as x,y pairs seperated by spaces - i.e: "3,4 5,9 11,3"

You can generate points for a given secret using the secret encoder.
